## 📝 입력 방법

### 1️⃣ 제목
```

```

### 2️⃣ 본문
```
💪 1.5h
🗣️ 45m
🔬 3h - VQE 회로 최적화 실험
📚 Quantum Computing - Ch.3 양자 게이트
```

### 📌 시간 입력 형식
- `1h` 또는 `1시간` → 1시간
- `30m` 또는 `30분` → 30분
- `1.5h` → 1시간 30분
