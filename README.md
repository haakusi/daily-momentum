# 🎯 Daily Momentum

> 매일매일 조금씩, 꾸준히 나아가는 PhD 여정 🚀

## 🎮 시작하기

### 1️⃣ 첫 기록 남기기
1. [New Issue](../../issues/new/choose) 클릭
2. "📝 Daily Log" 템플릿 선택
3. 오늘의 활동 입력
4. Submit!

### 2️⃣ 입력 형식
```
💪 1.5h
🗣️ 45m
🔬 3h - VQE 회로 최적화 실험
📚 Quantum Computing - Ch.3 양자 게이트
```

### 📝 입력 예시

**간단 버전:**
```
💪 1h
🗣️ 30m
🔬 2h
```

**상세 버전:**
```
💪 1.5h - 가슴/삼두 운동
🗣️ 45m - 영어 회화 연습 (주제: 연구 발표)
🔬 3h - VQE 회로 최적화 실험, 논문 초안 작성
📚 Quantum Computing: An Applied Approach - Ch.3 양자 게이트 정리
```

### 3️⃣ 자동으로 처리되는 것들
- ✅ 주간/월간/연간 로그 자동 생성
- ✅ 통계 자동 계산 및 시각화
- ✅ 대시보드 자동 업데이트
- ✅ 독서 기록 자동 정리
- ✅ Issue 자동 닫기

## 🤝 커뮤니티 공유

이 시스템은 누구나 복사해서 사용할 수 있습니다!

### 포크하는 방법
1. 이 레포 Fork
2. 자신의 목표에 맞게 커스터마이징
3. Issue로 기록 시작!

### 같이 동기부여하기
- 레포를 Public으로 설정하면 서로의 진행상황 확인 가능
- Star를 눌러 응원하기
- 자신만의 커스터마이징을 Issues/Discussions에서 공유
